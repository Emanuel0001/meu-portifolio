GeraisNet - Site de Marketing Digital

O projeto "GeraisNet" foi uma colaboração direta entre mim, como desenvolvedor, e o proprietário da empresa provedora de internet. Iniciamos o projeto com uma comunicação aberta e frequente, o que nos permitiu compreender completamente as necessidades e requisitos do site. Um dos principais desafios era a apresentação dos preços dos planos de internet de forma clara e acessível aos visitantes do site. Com base nessas discussões, conseguimos criar um site que atendesse às necessidades específicas da empresa e oferecesse uma experiência do usuário intuitiva ...

No que diz respeito às tecnologias utilizadas, optamos por construir o site "GeraisNet" com Next.js, aproveitando suas vantagens de renderização do lado do servidor e desempenho otimizado. Além disso, incorporamos JavaScript, HTML e CSS para a criação de recursos interativos e um design atraente. Também cuidei do registro de domínio, o que incluiu a seleção do nome de domínio apropriado e a compra, garantindo que o site estivesse acessível online para todos os usuários em potencial. O resultado foi um site eficiente e funcional que atendeu às expectativas da empresa e seus clientes.

PizzaApp - Site de Cardápio Digital

O projeto "PizzaApp" foi desenvolvido em estreita colaboração com o proprietário de uma pizzaria local, visando criar um cardápio online inovador. A comunicação direta com o dono da empresa desempenhou um papel fundamental, permitindo-nos compreender suas necessidades e requisitos específicos. Uma das principais demandas era proporcionar uma experiência de acesso mais fácil para os clientes, e para isso, implementamos a funcionalidade de QR code, permitindo que os clientes digitalizassem facilmente o código para acessar o cardápio online, simplificando assim a sua experiência ...

Quanto às tecnologias utilizadas, optamos por construir o "PizzaApp" com ReactJS, aproveitando sua flexibilidade e eficiência no desenvolvimento de interfaces de usuário dinâmicas. Além disso, incorporamos JavaScript, HTML e CSS para criar uma interface amigável e atrativa para os clientes. Para manter os custos baixos, não optamos pelo registro de um domínio personalizado, em vez disso, utilizamos um domínio gratuito para garantir que o cardápio online estivesse acessível a todos. O resultado foi um projeto bem-sucedido que atendeu às necessidades do clientes da pizzaria e ofereceu uma experiência conveniente aos clientes.

Finance.com - Site Próprio

No projeto "Finance Control," como desenvolvedor, priorizei a implementação de um sistema de controle de gastos robusto e seguro. Para alcançar isso, integrei um banco de dados online que permitiu que cada usuário tivesse seu próprio login e armazenasse seus dados de maneira segura. Essa abordagem garantiu que os usuários pudessem acessar suas informações financeiras pessoais de qualquer lugar, ao mesmo tempo em que mantinham seus dados protegidos e organizados ...

Quanto às tecnologias empregadas no projeto, escolhi ReactJS, aproveitando sua capacidade de criar interfaces de usuário dinâmicas e responsivas. Além disso, utilizei JavaScript, HTML e CSS para criar uma experiência de usuário amigável e atraente. Para hospedar o site, optei pelo Netlify, uma escolha econômica que permitiu que o projeto estivesse disponível online sem a necessidade de adquirir um domínio pago. Essa abordagem tornou o "Finance Control" acessível a um público amplo, proporcionando uma solução eficaz para o controle de gastos pessoais.

Sistema de Usuários - Site Próprio

O projeto "Sistemas de Usuário" foi uma experiência de estágio significativa para o meu desenvolvimento como programador. Durante esse estágio, fui orientado por dois programadores experientes que me ensinaram as nuances do desenvolvimento de aplicativos web usando tecnologias como ReactJS, Express, JavaScript, CSS, Git e GitHub. Tive a oportunidade de participar de reuniões diárias nas quais discutíamos o que eu estava estudando e aplicando em tempo real, o que proporcionou um aprendizado prático e eficaz ...

Em relação às características do projeto, ele abrange uma série de funcionalidades essenciais para o gerenciamento de usuários. O sistema inclui uma página de cadastro e login de usuários para autenticação segura. No painel principal, há um dashboard que exibe uma lista completa de todos os usuários do sistema, e a adição de uma foto de perfil para cada usuário é uma das funcionalidades destacadas. Essas características fornecem uma experiência abrangente para o usuário e demonstram as habilidades aprendidas durante o estágio de forma prática e funcional.
